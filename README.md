# Canvas & Graphics Skills

Production-ready AI agent skills for canvas rendering, SVG manipulation, and PDF generation.

## Skills

| Skill | Description |
|-------|-------------|
| **fabricjs-canvas** | Fabric.js framework + HTML5 Canvas 2D API — objects, events, serialization, caching, transforms, clipPath, filters, text, groups, zoom/pan, controls, known issues |
| **svg-editor** | SVG creation & manipulation based on W3C SVG 2 spec — paths, viewBox, gradients, patterns, clip paths, masks, filters, text layout, accessibility |
| **pdf-guru** | PDF specification expert (ISO 32000-1:2008 / PDF 1.7) — document structure, content streams, operators, fonts, encryption, transparency, annotations, forms |

## Install

```bash
npx skills add hwhovo/canvas-skills
```

Or install individual skills:

```bash
npx skills add hwhovo/canvas-skills/fabricjs-canvas
npx skills add hwhovo/canvas-skills/svg-editor
npx skills add hwhovo/canvas-skills/pdf-guru
```
